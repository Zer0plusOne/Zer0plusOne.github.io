# Herramientas

En esta sección documento herramientas de uso frecuente en ciberseguridad.

## Estructura recomendada

Para cada herramienta:

1. Nombre y versión.
2. Casos de uso.
3. Comandos base.
4. Notas de hardening y OPSEC.
5. Errores comunes y solución.

## Categorías iniciales

- Reconocimiento
- Enumeración
- Explotación controlada
- Post-explotación en laboratorio
- Forense y análisis
